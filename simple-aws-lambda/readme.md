
# Install and configure

* Node
* NPM
* AWS cli
* Serverless framework
$ npm install -g serverless

## Config AWS Credentials

$ serverless config credentials --provider aws --key <AWSAccessKeyId> --secret <AWSSecretKey> --profile serverless-admin
